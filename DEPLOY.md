# Деплой PrismaLab

## Команда деплоя

```bash
cd ~/PrismaLab && ./deploy.sh
```

Скрипт копирует проект на сервер, запускает миграции и поднимает контейнеры.

---

## Что изменилось (Mini App)

### Главный экран
- Убрана плашка «Экспресс-фото»
- Заголовок «PrismaLab» с анимацией
- Текст: «Добро пожаловать в студию PrismaLab. В каждом фотосете мы даём сразу несколько фотографий, чтобы вы могли выбрать лучшее»
- Одна плашка: «Готовые фотосеты»

### Категории паков
- Вверху экрана: **Женские** | **Детские** | **С животными**
- Категория определяется автоматически по ID пака (248, 682 — животные; 593, 859, 2152, 2501 — детские; остальные — женские). Менять `.env` не нужно.

### Пол
- Спрашиваем только если не знаем (нет в профиле)
- При выборе в Mini App сохраняем в профиль (как в боте)

## Если менял .env

`.env` не копируется деплоем. Обнови вручную:

```bash
scp .env root@194.87.133.7:/root/PrismaLab/
```

Потом перезапусти:

```bash
ssh root@194.87.133.7 "cd /root/PrismaLab && docker compose up -d --force-recreate prismalab"
```
